# Apostila - Utilização de Software de Criação e Edição de Vídeo

Apostila ilustrada em HTML desenvolvida para o curso **Marketing e Gestão de Mídias Digitais** da **Universidade Federal do Norte do Tocantins (UFNT)**.

## Acessar a apostila

Abra a versão completa em HTML:

https://htmlpreview.github.io/?https://github.com/bibliodados/Apostila-MidiasDigitais/blob/main/index.html

## Tema

**Fluxo de trabalho da produção de vídeo** aplicado ao marketing digital.

A apostila aborda as principais etapas do processo:

- Briefing
- Roteiro
- Captação
- Edição
- Revisão
- Exportação
- Publicação
- Análise

## Arquivo do site

- `index.html`

## Uso

O arquivo HTML possui layout em formato A4 e botão para **salvar em PDF / imprimir**.
